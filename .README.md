### ℹ️ Description

Din Conquest is a 2D top-down RPG built with Unity.

The project aims to create an immersive educational adventure game focused on learning authentic Islamic knowledge through quests, exploration, and character interaction.

Current development status:

- ✅ Player movement (4 directions) with Blend Tree animations
- ✅ Last direction state preserved while idle
- ✅ Collision system with environment
- ✅ Asset integration (Arabian medieval tileset & sprites)
- 🔄 NPC random roaming system in progress
- 🔄 Interaction system (NPC turns toward player when engaged)

The long-term goal is to build a structured quest-based adventure inspired by classic RPGs, combining storytelling, exploration, and beneficial knowledge.

---

## 🚀 Planned Features

- 🧍 Intelligent NPC wandering within defined walkable zones
- 🧭 Quest system with structured objectives
- 💬 Dialogue system with branching interactions
- 🏘️ Interactive city with buildings and interiors
- 📜 Educational challenges integrated into quests
- 🎵 Ambient city sound design
- 🗺️ Multiple explorable maps
- 💾 Save & load system
- 🌐 WebGL build for online publishing

# ✅ Getting Started

This project runs on 🎮 Unity (2D URP / Built-in depending on version).

Below are installation instructions for running the project locally.

### 💻 Installation

```bash
# 1. Clone the repository
git clone https://github.com/nlabrazi/din-conquest.git

# 2. Open Unity Hub
# 3. Add project from disk
# 4. Select the project folder
# 5. Open using the recommended Unity LTS version
```

Make sure you have:
- Unity LTS installed
- 2D Sprite package enabled
- Input System configured (if used)
